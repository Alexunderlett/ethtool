# 账户管理

理解私钥、公钥和地址的关系，学习离线账户的创建与载入。

## 创建新账户

在2#终端进入`~/repo/chapter3/new-account`目录并运行Go程序：

```
~$ cd ~/repo/chapter3/new-account
~/repo/chapter3/new-account$ go run main.go
```
